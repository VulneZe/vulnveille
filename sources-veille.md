## Notes d'intégration

### FreshRSS
Ajouter les flux RSS dans FreshRSS avec les catégories correspondantes.

### changedetection.io
Ajouter les pages sans RSS dans changedetection.io avec une fréquence de surveillance quotidienne ou hebdomadaire selon la criticité.

### n8n
Les workflows n8n peuvent utiliser ces sources pour :
- Générer des digests
- Envoyer des alertes critiques
- Surveiller les fins de support
